# TrayCharger

Windows 11 시스템 트레이에 노트북 배터리 충전전류를 실시간으로 표시합니다.

## 기능

- 충전전류를 트레이 아이콘에 숫자로 표시 (예: `2.3`)
- 충전 중 → 검정색, 방전 중 → 빨간색
- 마우스 오버 시 상세 정보 (mA, W, 전압, 배터리%)
- 우클릭 메뉴: 자동 실행 등록/해제, 트레이 설정, 종료
- 단일 exe (12KB), 별도 설치 불필요

## 요구사항

- Windows 10/11
- .NET Framework 4.x (Windows 기본 내장)
- 노트북 (배터리 필요)

## 빌드

```bat
build.bat
```

## 사용법

1. `TrayCharger.exe` 실행
2. 시스템 트레이에 충전전류 숫자 표시
3. 우클릭 → "시작시 자동 실행" 체크하면 부팅 시 자동 시작
4. 우클릭 → "트레이에 항상 표시 (설정)" 으로 아이콘 고정

## 표시 방식

| 상태 | 아이콘 | 색상 |
|------|--------|------|
| 충전 2.3A | `2.3` | 검정 |
| 방전 1.8A | `-1.8` | 빨강 |
| 완충 (AC 연결) | `0` | 검정 |

## 툴팁 (마우스 오버)

```
충전: 2300mA (36.8W)
전압: 16.1V
배터리: 33%
AC: 연결
```
